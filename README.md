# Basic Synthesizer

Cmajor で作成された、Webブラウザベースの GUI を持つシンプルな減算方式シンセサイザーです。

## 機能概要

- **オシレーター (Oscillator)**
  - 4種類の波形を選択可能: `Sine` (正弦波), `Saw` (ノコギリ波), `Square` (矩形波), `Triangle` (三角波)
- **エンベロープ (ADSR Envelope)**
  - 音量の時間的変化を制御します。
  - `Attack` (立ち上がり), `Decay` (減衰), `Sustain` (持続レベル), `Release` (余韻)
- **フィルター (Low Pass Filter)**
  - 音の明るさを調整します。
  - `Cutoff` (カットオフ周波数), `Resonance` (レゾナンス)
- **MIDI 入力対応**
  - Web MIDI API を使用して、接続された MIDI キーボードで演奏が可能。

## 使い方

### 1. 起動方法
このプロジェクトは VS Code の Cmajor 拡張機能を使用して実行します。

1.  VS Code で `BasicSynth.cmajorpatch` ファイルを開きます。
2.  エディタ右上の「再生ボタン (Play)」をクリックするか、コマンドパレットから `Cmajor: Play Patch` を実行します。
3.  Webブラウザが立ち上がり、シンセサイザーの UI が表示されます。

### 2. 画面操作

| セクション | パラメータ | 説明 |
| :--- | :--- | :--- |
| **Oscillator** | **Waveform** | 発音する波形の種類を選びます。 |
| **Envelope** | **Attack** | 鍵盤を押してから最大音量になるまでの時間。 |
| | **Decay** | 最大音量からサステインレベルに下がるまでの時間。 |
| | **Sustain** | 鍵盤を押し続けている間の音量レベル。 |
| | **Release** | 鍵盤を離してから音が消えるまでの時間。 |
| **Filter** | **Cutoff** | 高周波成分をカットする基準周波数。下げるとこもった音になります。 |
| | **Resonance** | カットオフ付近の周波数を強調し、癖のある音にします。 |
| **Frequency** | (Slider) | 周波数モニター（MIDI入力時は自動で動きます）。 |
| **Gate** | (Indicator) | 鍵盤が押されているかどうかの表示。 |
| **Master Volume** | (Slider) | 全体の音量調整。 |

### 3. 演奏方法
- **画面上のキーボード**: マウスで鍵盤をクリックして演奏できます。
- **PCキーボード**: 以下のキーマッピングで演奏可能です。
  - `A` `S` `D` `F` `G` `H` `J` `K` ... (白鍵: ド〜ド)
  - `W` `E` `T` `Y` `U` `O` `P` ... (黒鍵)
- **MIDIキーボード**: PCに接続された MIDI キーボードを弾くと発音します。
- **MIDI信号**: ブラウザが MIDI デバイスへのアクセスを求めた場合は「許可」してください。

## ファイル構成
- `BasicSynth.cmajor`: シンセサイザーの信号処理ロジック（オシレーター、フィルター、ADSRの実装）。
- `view/`: Web UI の構成ファイル (`index.html`, `style.css`, `main.js`)。
